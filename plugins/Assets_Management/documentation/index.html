Installation

Quick Installation Guide
  • Go to Settings > Plugins.
  • Install the Assets_Management_for_RISE_CRM.zip which you have downloaded from the Envato Market with a valid purchase code.
  • If everything is OK, it'll show a success message.
  • Activate the plugin from the option dropdown.

Assets Management settings

  • You'll get the Assets Management settings in Settings > Plugins > Assets Management.

  • Firstly, Please create Asset group, Asset unit and Asset Location from here.
  • Create a new Asset group from Add asset group button.


  • Create a new Asset unit from Add asset unit button.


  • Create a new Asset location from Add asset location button.

Upgrading Guide

Please note that you can only download the updates if you have a valid purchase code.

Please backup all files and database before start the installation of updates.

There is a very easy way to update your application.

To install any updates go to Settings>Plugins>Assets Management>Dropdown>Updates of the application. The download will be start automatically once any new update is available. Once the download is completed you can install the updates by clicking on the link. That’s all!
